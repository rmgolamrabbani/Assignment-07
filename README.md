📘 KeenKeeper

A simple React + Vite web application to manage friendships and track interactions like Call, Text, and Video. It includes a timeline system, relationship tracking, and real-time updates with notifications.

-------------------------------------------

🚀 Features

👤 Friend details page with profile info
📊 Relationship tracking (Days since contact, goal system)
📅 Dynamic Timeline system (Call / Text / Video logs)
🔍 Filter timeline by type
🔔 Toast notifications on actions
🔄 Real-time updates using React Context API
💾 Persistent data using LocalStorage
🔙 Back navigation support
🎨 Clean and responsive UI with Tailwind CSS
⚡ Smooth loading experience (spinner / route loader)


--------------------------------------------


🛠️ Tech Stack

React.js
Vite
React Router DOM
Context API
Tailwind CSS
React Icons
React Hot Toast


---------------------------------------

📌 How it works

👤 Friend Details Page
Shows friend info
Calculates relationship status:
On-Track
Almost Due
Overdue

-------------------------------------------

📅 Timeline System

Click Call / Text / Video
Automatically adds entry to timeline
Stored globally using Context API
Saved in LocalStorage

-----------------------------------------

🔔 Notifications
Every action triggers a toast message
Example:
"Call added to timeline ✅"
